Awesome Window & Tab Manager

The Awesome Window & Tab Manager allows you to manage all Chrome windows and tabs.

Beautiful, I know. But there is a lot going on. Let's break it down:

(1) - This is how many windows and tab are currently open. It the example, 7 windows and 47 tabs.

(2) - This is the filter box. Type in a text filter, and the windows and tabs are reduced to those that match.

Show Example

When typing in the filter box, the tabs are filtered. For details in how filtering works, see 'Options'.



(3) - Alpha Sort - check this box and the tabs are sorted alphabetically. Otherwise, they are presented in the same order they appear in Chrome.

(4) - Show - select whether to display the tab title or the URL.

(5) - Categorize - by default (and in the screenshot) the tabs are separated by window. Alternatively, they can be separated by domain, category, or not all all.

(6) - All windows and tabs can be saved, and thus restored later.

Show Example

You can save sets of windows and tabs as a Session.

(1) - Type a name for the session here.

(2) - Click here to save the session.

(3) - Previously saves sessions are listed here. Click the name or the icon to restore the session.

(4) - Click the 'x' to delete a previously saved session.

(5) - Check this box to have all other Chrome windows and tabs closed when restoring a session.



(7) - All tabs in a window, domain, or group can be refreshed.

(8) - All tabs in a window, domain, or group can be saved, and thus restored later.

(9) - All tabs in a window, domain, or group can be closed.

(10) - Any individual tab can be closed.

(11) - This book icon just means the tab is bookmarked already.

(12) - This plus icon is used to add a bookmark for this tab.

Show Example

Bookmarks can be added for tabs.

(1) - The title of the page from the tab. Can be optionally altered before saving.

(2) - The URL of the page from the tab. Can be optionally altered before saving.

(3) - Click a bookmark folder to add the new bookmark.



(13) - Clicking the icon or the title takes you to that window & tab.
